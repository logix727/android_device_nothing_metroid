# Security Policy

## Supported Scope

This repository contains device-tree source configuration for Nothing Phone (3) (`metroid`).

## Reporting A Vulnerability

If you find a security issue affecting this repository:

1. Do not open a public issue with exploit details.
2. Open a private GitHub security advisory draft for this repository, if available.
3. If private reporting is unavailable, open a minimal issue requesting a secure contact path.

Please include:

- Affected file path(s)
- Impact summary
- Reproduction steps
- Suggested mitigation (if known)

You can expect an initial response as soon as practical.
